# Калькулятор оценок Platonus

Приложение для расчета оценок в системе Platonus. Позволяет рассчитывать оценки для РК1, РК2, итоговой оценки и GPA.

## Функциональность

- **Калькулятор РК1**: Рассчитывает сумму баллов и среднюю оценку для первого рубежного контроля.
- **Калькулятор РК2**: Рассчитывает сумму баллов и среднюю оценку для второго рубежного контроля.
- **Калькулятор итоговой оценки**: Рассчитывает итоговую оценку на основе экзамена, РК1 и РК2.
- **Калькулятор GPA**: Рассчитывает GPA на основе введенных предметов, оценок и кредитов.
